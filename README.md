# Sample Regression

A foundational machine learning repository demonstrating a standard regression pipeline. 

## Current Implementation
- **Language/Libraries:** Python, Pandas, NumPy, Scikit-Learn
- **Model:** Multiple Linear Regression
- **Data:** Synthetic generated dataset (5 features)
- **Metrics Evaluated:** Mean Squared Error (MSE), R-squared ($R^2$)

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Execute the pipeline: `python regression_pipeline.py`
